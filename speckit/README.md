## To Install SpecKit

See instructions in

https://github.com/github/spec-kit

**TL;DR**

```shell
# Install
uv tool install specify-cli --from git+https://github.com/github/spec-kit.git

# Upgrade
uv tool install specify-cli --force --from git+https://github.com/github/spec-kit.git
```
